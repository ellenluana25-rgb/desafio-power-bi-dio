# 📊 Processando e Transformando Dados com Power BI

Projeto desenvolvido como parte do desafio da plataforma **Digital Innovation One (DIO)**.

---

## 🎯 Objetivo
Aplicar técnicas de **tratamento, transformação e modelagem de dados** utilizando o **Power BI**, transformando dados brutos em informações úteis para análise.

---

## 🛠️ Ferramentas Utilizadas
- Microsoft Power BI
- Power Query
- Microsoft Excel

---

## 🔄 Etapas do Projeto

✔️ Importação dos dados  
✔️ Limpeza e tratamento de inconsistências  
✔️ Transformação dos dados no Power Query  
✔️ Modelagem dos dados  
✔️ Criação de visualizações e dashboard interativo

---

## 📈 Resultado

Foi desenvolvido um **dashboard interativo** que permite visualizar e analisar os dados após o processo de transformação, facilitando a interpretação das informações e a tomada de decisão.

---

## 🚀 Aprendizados

Durante o desenvolvimento deste projeto foi possível praticar:

- Transformação de dados no Power Query  
- Modelagem de dados  
- Criação de dashboards interativos  
- Boas práticas de análise de dados
